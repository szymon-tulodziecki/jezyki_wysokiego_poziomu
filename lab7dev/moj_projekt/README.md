## Opis
Aplikacja blogowa o jedzeniu wykorzystująca:
- Next.js (App Router)
- React Server Components
- CSS Modules
- Image Optimization
- Dynamic Routing

## Struktura projektu
- `/meals` - przeglądanie posiłków
- `/meals/share` - udostępnianie posiłków
- `/meals/[mealSlug]` - dynamiczne route dla szczegółów posiłku
- `/community` - społeczność foodie

## Technologie
- Next.js 14+
- React 18+
- CSS Modules
